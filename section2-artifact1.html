Artifact 1: Blended Learning Module — Technology-Enhanced STEM Instruction

Rationale

This artifact is a blended learning module developed during the MA EDIN program that integrates face-to-face instruction with digital learning platforms to create a transformative STEM education experience. The module was designed using the SAMR (Substitution, Augmentation, Modification, Redefinition) model (Puentedura, 2014) to ensure that technology integration moves beyond simple substitution toward genuine redefinition of the learning experience. The module incorporates interactive simulations, collaborative online workspaces, formative digital assessments, and student-created multimedia projects that allow learners to demonstrate their understanding through multiple modalities.

This artifact was selected for the portfolio because it directly demonstrates PLO 2—the ability to practice instructional methods and technologies for transformative learning. The blended learning design addresses a critical challenge in contemporary education: how to leverage technology not merely as a delivery mechanism but as a catalyst for deeper, more meaningful learning (Garrison & Vaughan, 2008). By combining synchronous collaborative activities with asynchronous self-paced learning, the module accommodates diverse learner needs while maintaining the rigor and engagement necessary for effective STEM instruction. The accompanying assessment rubric demonstrates competency in using evaluation methods to determine the effectiveness of the instructional program (PLO 2c).
